# 🔐 React Node.js MySQL Authentication App

A full-stack authentication project built with:

- **Frontend:** React (Vite) + Tailwind CSS  
- **Backend:** Node.js + Express  
- **Database:** MySQL  
- **Authentication:** JWT (JSON Web Token)

---

## 📁 Project Structure

project:
  backend:
    routes:
      - authRoutes.js
    lib:
      - db.js
    - index.js
    - .env

      pages:
        - Login.jsx
        - Register.jsx
        - Home.jsx
      components:
        - Navbar.jsx
